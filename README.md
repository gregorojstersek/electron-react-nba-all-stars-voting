# NBA All Stars Voting 2019

## Features:

Vote on Google and nba.com with pre made selection handled by Puppeteer and Chromium.

## Technologies:

Electron, React, Webpack, ES8, Puppeteer and others

## Installation:

```bash
# Install dependencies
yarn

# Development
yarn dev

# Production build
yarn package
```
